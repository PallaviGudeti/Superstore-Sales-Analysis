**Superstore Sales Anlaysis**

This project focuses on analyzing retail sales data from the Global Superstore dataset. The goal wast to uncover key business insights such as regional sales performance, product profitability using Excel and Power BI.

**Dataset:**
 This dataset contains 21 columns.

**Source:** Kaggle

**Sheet used:** Orders

**Tools Used:**
- **Microsoft Excel**
  - Power Query (data cleaning)
  - Pivot Tables & Charts (pre-analysis)
- **Power BI**
  - Dashboard creation
  - Data modeling & interactive visualizations

**Skills Applied:**
1. Data Cleaning & Transformation
2. Data Visualization
3. Business Intelligence
4. Dashboard Design
5. Excel + Power BI Workflow

**Key Features:**
- Cleaned and structured raw sales using Excel Power Query.
- Created pivot tables to explore sales by month, region, category, and discount.
- Built an interactive Power BI dashboard with:
  - Sales trends over time
  - Sales & profit by region, category, and sub-category
  - Sales by segment
  - State-wise sales on a U.S. map
- Slicers for Region, Segment, and Category

**Business Insights:**
- The West region generated the highest total sales.
- Technology category had the highest profit margin.
- Higher discounts often correlated with lower profit.
- The Consumer segment was the most profitable.
